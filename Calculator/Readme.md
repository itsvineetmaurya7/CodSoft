
# Simple Calculator

A lightweight web-based calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division, and percentage
- Decimal support
- Clear (`C`) button to reset the input
- Delete (`DEL`) button to remove the last entered character
- Responsive, modern UI with a clean calculator layout

## Files

- `index.html` - the main page structure and calculator buttons
- `style.css` - styling for layout, buttons, and display
- `script.js` - calculator logic for input handling and evaluating expressions

## Usage

1. Open `index.html` in your web browser.
2. Click the buttons to enter numbers and operations.
3. Press `=` to calculate the result.
4. Use `C` to clear the display or `DEL` to delete the last character.

## Notes

- This project uses JavaScript's `eval()` function to evaluate expressions. It is intended for a simple local calculator demo.
- Keep input limited to valid calculator expressions to avoid errors.
