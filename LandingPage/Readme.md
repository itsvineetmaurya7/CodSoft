# FRIENDS Restaurant Landing Page

A responsive restaurant landing page built with HTML and CSS. It features a hero section, menu highlights, customer reviews, and contact information.

## Features

- Full-page hero section with a restaurant brand and call-to-action
- Navigation menu with smooth scrolling to page sections
- About section describing the restaurant
- Featured dishes and menu item cards with images and prices
- Customer reviews section
- Contact details in the footer
- Responsive design for smaller screens

## Files

- `index.html` – HTML structure and page content
- `style.css` – Styling for layout, hero section, menu cards, and responsiveness
- `images/` – Photos used in the landing page

## Usage

1. Open `index.html` in your web browser.
2. Scroll through the page to view the about section, menu, reviews, and contact details.
3. Use the navigation links to jump to different sections of the page.

## Notes

- The page is built using pure HTML and CSS with no JavaScript.
- Replace the placeholder images in `images/` with your own restaurant photos if desired.
