# Codsoft Repository

Welcome to the **Codsoft** repository. This repo contains three small web development projects built with HTML, CSS, and JavaScript.

## Projects Included

### 1. Calculator
- Location: `calculator/`
- A simple web-based calculator.
- Includes functionality for basic arithmetic operations like addition, subtraction, multiplication, and division.
- Live Demo : https://itsvineetmaurya7.github.io/CodSoft/Calculator/
- Files:
  - `index.html`
  - `style.css`
  - `script.js`

### 2. Landing Page
- Location: `LandingPage/`
- A responsive landing page layout.
- Designed to showcase a product or service with clean visual sections.
- Files:
  - `index.html`
  - `style.css`
  - `Readme.md`
  - `images/` (assets used by the landing page)
- Live Demo : https://itsvineetmaurya7.github.io/CodSoft/LandingPage/

### 3. Portfolio Project
- Location: `Portfolio Project/`
- A portfolio website to present personal work and skills.
- Contains a homepage layout, project sections, and styling for a professional look.
- Files:
  - `index.html`
  - `style.css`
  - `README.md`
  - `Asset/` (supporting images or files)
- Live Demo : https://itsvineetmaurya7.github.io/CodSoft/Portfolio/

## How to Use
1. Open the project folder you want to view.
2. Open the `index.html` file in a browser.
3. Interact with the page or use the interface as intended.

## Notes
- Each project is self-contained and can be opened independently.
- The repository is intended as a showcase of basic front-end development skills.
